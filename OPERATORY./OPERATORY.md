# OPERATORY

## Przegląd operatorów

| Operator | Funkcja | Przykłady pol. |
|----------|--------|----------------|
| K-       | pytajny | kto, gdzie, kiedy |
| J-       | względny / anaforyczny | jego, jej |
| T-       | wskazanie bliskie | ten, tu, teraz |
| IN-      | alternatywa / inność | inny, inaczej |
| OW-      | wskazanie pośrednie / anafor. | ów, owędy |
| ON-      | wskazanie dalekie / os. trzecia | on, ona |
| WSZ-     | wskazanie wszystkich opcji | wszystko, wszelakie |

---

## Końcówki funkcjonalne

| Końcówka | Kategoria | Znaczenie |
|----------|----------|-----------|
| -o       | nijaki zaimek | to/ten/ta |
| -ako     | przysłówek sposobu | tak |
| -aki/-aka/-akie | przymiotnik jakości | taki |
| -ędy     | przysłówek drogi | tędy |
| -eda     | przysłówek czasu | wtedy |
| -liko    | przysłówek ilości | tyle |
| -edy     | przysłówek miejsca | tam |
| -mo      | przysłówek miejsca (kierunkowy) | tam / kierunek |

---

### Uwagi

- **-aki** = -a + -ki (cecha/jakość)
- **-ako** łączy się z liczebnikami:  
  *jednako, dwojako, dwojaki*
- **-mo** łączy się z rdzeniami czasowników (ruch/kierunek)

---

## K- — pytajny

| Forma | Znaczenie |
|-------|----------|
| ko        | który |
| kako      | jak? |
| kaki/kaka/kakie | jaki? |
| kędy      | którędy |
| kieda     | kiedy |
| koliko    | ile |
| kedy      | jakie miejsce |
| kamo      | dokąd |

**Przedrostki:**
- nie- → nieokreśloność (*niekiedy*)
- ni- → negacja (*nikt, nigdy, nigdzie*)

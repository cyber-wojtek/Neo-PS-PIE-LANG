# 🔷 PRZYROSTKI

## I. PRZYROSTKI TWORZĄCE PRZYMIOTNIKI

### -y / -i
**Funkcja:** adiektywizacja rdzenia (cecha wynikająca z bazy leksykalnej)  
**Znaczenie:** „taki, który posiada cechę X / jest związany z X”  
**Typ:** denominalny, deverbatywy  
**Produktywność:** bardzo wysoka — rdzeń każdego przymiotnika prymarnego  

**Przykłady:**
- biały → „mający cechę bieli” (psł. *bělъ)
- zły → „mający cechę zła / będący złym” (psł. *zьlъ)
- nowy → „mający cechę nowości” (psł. *novъ)
- młody → „mający cechę młodości” (psł. *moldъ)

### -ki / -oki / -eki / -aki
**Funkcja:** tworzenie cechy wyodrębnionej z rdzenia, często z odcieniem intensywności lub kierunkowości  
**Znaczenie:** „posiadający cechę X w sposób określony / uchwytny”  
**Typ:** denominalny  

**Przykłady:**
- miękki → „mający miękkość” (psł. *mękъkъ)
- gładki → „mający gładkość” (psł. *gladъkъ)
- rzadki → „mający rzadkość” (psł. *rědъkъ)
- słodki → „mający słodkość” (psł. *sladъkъ)
- głęboki → „sięgający w głąb” (psł. *golbokъ)
- szeroki → „rozciągający się wszerz” (psł. *širokъ)
- wysoki → „sięgający ku górze” (psł. *vysokъ)

### -ow-
**Funkcja:** relacja / przynależność / pochodność; tworzenie przymiotników relacyjnych  
**Znaczenie:** „należący do X, przynależny do X, spadkobierca X”  
**Typ:** denominalny — od rzeczowników  

**Przykłady:**
- rodowy → „związany z rodem”
- Ivanow (ros.) → „należący do Ivana”
- dębowy → „związany z dębem / zrobiony z dębu”
- złoty → (przez -t-ow-) „związany ze złotem”
- stalowy → „wykonany ze stali”
- domowy → „związany z domem”
- morski (przez -sk-) → „związany z morzem”

### -n-
**Funkcja:** materiałowość / źródłowość / związanie  
**Znaczenie:** „wykonany z X / dotyczący X”  
**Typ:** denominalny i deverbatywy  

**Przykłady rzeczownikowe (denominalny):**
- ziemny → „z ziemi / ziemisty” (ziemia + -n-)
- smutny → „napełniony smutkiem”
- wiosenny → „dotyczący wiosny”
- letni → „dotyczący lata”
- nocny → „dotyczący nocy”
- środkowy → „dotyczący środka / centrum” (-k-ow-)

**Przykłady czasownikowe (deverbatywy):**
- wolny → „ten, kto może woleć / nie jest przywiązany” (< vol- + -n-)
- ważny → „mający wagę / ciężar decyzyjny”
- ważny → „ważący się; dotyczący wagi”
- czytelny → „nadający się do czytania” (czyt- + -el-n-)

### -ist- / -yst- / -ast-
**Funkcja:** intensyfikacja cechy / obfitość / podobieństwo do X  
**Znaczenie:** „obfitujący w X / pełen X / mający wiele X / podobny do X”  
**Typ:** denominalny  
**Geneza:** psł. *-istъ / *-astъ; paralele w łacinie *-ōsus i germańskim *-iska-; spokrewniony też z gr. -ίσκος  

**Przykłady:**
- piaszczysty → „pełen piasku”
- krzaczasty → „pełen krzaków”
- mglisty → „pełen mgły”
- skalisty → „pełen skał / jak skała”
- ziarnisty → „mający ziarna / pełen ziaren”
- liściasty → „mający dużo liści”
- kudłaty → „mający kudły”
- rogaty → „mający rogi”
- brodaty → „mający brodę”

### -sk-
**Funkcja:** przynależność do grupy, miejsca, narodu, osoby, zawodu  
**Znaczenie:** „należący do X / będący z X / właściwy dla X”  
**Typ:** denominalny  
**Geneza:** psł. *-ьskъ, jedno z najproduktywniejszych  

**Przykłady:**
- polski → „z Polski / dotyczący Polaków”
- krakowski → „z Krakowa”
- rycerski → „należący do rycerza / jak rycerz”
- wiejski → „dotyczący wsi”
- morski → „dotyczący morza”
- ludzki → „dotyczący ludzi”
- Bożyński, Kowalski → „należący do Boga / kowala”

### -liw-
**Funkcja:** skłonność / skłonność do czynności, gotowość do przejawu cechy  
**Znaczenie:** „mający skłonność do X / skłonny do X”  
**Typ:** deverbatywy (od czasownika)  
**Geneza:** psł. *-livъ  

**Przykłady:**
- mściwy → „skłonny do mszczenia się”
- zazdrosny — ale: zazdrościwy → „skłonny do zazdrości”
- drażliwy → „łatwy do zdrażnienia”
- łagodliwy → „mający skłonność do łagodności”
- pojętliwy → „łatwo pojmujący”
- zapalczywy → „skłonny do zapalania się / gniewu”
- troskliwy → „pełen troski”

### -owat-
**Funkcja:** podobieństwo osłabione / niepełne posiadanie cechy  
**Znaczenie:** „podobny do X / trochę jak X / mający coś z X”  
**Typ:** denominalny  

**Przykłady:**
- brodawkowaty → „podobny do brodawki w kształcie”

### -awy (psł. *-avъ ← PIE -owos)
**Funkcja:** przymiotnik naturalnej skłonności / zabarwienia / przynależności  
**Znaczenie:** „mający naturalną właściwość X / zabarwiony X / skłonny do X z natury”  
**Typ:** denominalny  
**Geneza:** psł. -avъ ← PIE -owos — ablautowy wariant tego samego morfemu co -ovъ (-owy); podczas gdy -owy wyraża relację do X, -awy wyraża naturalne zabarwienie lub skłonność.  
**Uwaga:** nie mylić z -owat- — -owat- = „trochę podobny do X”, -awy = „mający naturalną właściwość X”.  

**Przykłady:**
- krwawy → „mający krew / naznaczony krwią” (krew + -awy)
- łaskawy → „mający łaskę / z natury łaskawy” (łaska + -awy)
- ciekawy → „mający ciekawość / płynący ciekiem zainteresowania” (ciek- + -awy)
- głodnawy (dial.) → „trochę głodny”
- brudnawy → „mający brud / trochę brudny”

### -al-n- / -el-n-
**Funkcja:** możliwość / zdolność / przeznaczenie  
**Znaczenie:** „mogący być X-owany / zdolny do X / dotyczący X”  
**Typ:** deverbatywy  

**Przykłady:**
- czytelny → „mogący być czytanym”
- widzialny → „mogący być widzianym”
- słyszalny → „mogący być słyszanym”
- palny → „zdolny do palenia się”
- zmywalny → „dający się zmyć”
- dostosowalny → „dający się dostosować”

### -czy (psł. *-čьjь ← PIE *-kyo-)
**Funkcja:** przymiotnik przynależności do dziedziny / roli agentywnej  
**Znaczenie:** „właściwy dla X / należący do dziedziny X / typowy dla roli X”  
**Typ:** denominalny (od nomina agentis na -nik, -acz, -ca)  
**Geneza:** psł. *-čьjь — złożenie -čь (palatalizacja z -kъ) + -jь (przymiotnikowy formant przynależności)  

**Przykłady:**
- twórczy → „właściwy twórcy / należący do dziedziny tworzenia” (twórca + -czy)
- pomocniczy → „właściwy pomocy / pomocnikowi” (pomocnik + -czy)
- robotniczy → „właściwy robotnikowi / klasie robotniczej” (robotnik + -czy)
- sądowniczy → „właściwy dziedzinie sądownictwa” (sądownik + -czy)
- wykonawczy → „właściwy wykonawcy” (wykonawca + -czy)
- ustawodawczy → „właściwy ustawodawcy” (ustawodawca + -czy)

### -ły (psł. -lъ)
**Funkcja:** nomen resultativum — ten/to który wykonał czynność i nosi jej rezultat w sobie  
**Znaczenie:** „ten, który X-ał i jest tym naznaczony / ten w którym czynność trwa jako stan”  
**Typ:** deverbatywy  
**Geneza:** psł. -lъ ← PIE -lo- — ten sam element który dał słowiański czas przeszły (robił, szedł, niósł).  

**Przykłady:**
- słaby ← -ły → słabły → „ten który osłabł / w którym osłabnięcie trwa”
- zgniły → „ten który zgnił i jest tym stanem”
- uschły → „ten który usechł”
- dojrzały → „ten który dojrzał i jest tym naznaczony”
- dorosły → „ten który dorósł — i jest tym na stałe”
- głupiały → „ten który zgłupiał”
- oniemiały → „ten który oniemiał”

### -ny / -ony / -ty (psł. *-nъ / -enъ / -tъ)
**Funkcja:** participium passivi — imiesłów bierny / przymiotnik rezultatywny  
**Znaczenie:** „ten, któremu wykonano czynność X / będący w stanie po wykonaniu X / naznaczony przez X”  
**Typ:** deverbatywy (od tematu czasownika)  
**Geneza:** PIE *-no- / *-to- — dwa odrębne sufiksy PIE które w psł. rozwinęły się w komplementarną dystrybucję  

**Przykłady:**
- robiony → „ten, którego ktoś robi”
- mazany → „ten, którego ktoś maże”
- szyty → „ten, którego ktoś szyje”
- cięty → „ten, którego ktoś tnie”
- dany → „ten, którego ktoś daje”
- rodzony → „ten, którego ktoś rodzi”

**Uwaga:** morfologicznie spokrewniony z -ły (psł. -lъ) — oba tworzą rezultatywy, ale:  
- -ły = rezultat aktywny („ten który X-ał i nosi to w sobie”)  
- -ny/-ty = rezultat bierny („ten któremu X-ano”)

## II. PRZYROSTKI TWORZĄCE RZECZOWNIKI

### A. Kategoria sprawcy / wykonawcy / nosiciela cechy

### -nik / -nik- / -nica / -nica-
**Funkcja:** nomen agentis — wykonawca, nosiciel, ten związany z czymś  
**Znaczenie:** „ten, kto X / ten, który zajmuje się X / ten, który posiada X”  
**Typ:** denominalny  
**Geneza:** psł. *-nikъ  

**Przykłady:**
- ogrodnik → „ten, kto zajmuje się ogrodem”
- myśliwiec (myśl-iw-ec / myśl-nik) → „ten, kto myśli lub poluje”
- strażnik → „ten, kto strzeże”
- dwornik → „ten, kto obsługuje dwór”
- więzień (więź-nik) → „ten, kto jest w więzi”
- robotnik → „ten, kto wykonuje robotę”
- lotnik → „ten, kto obsługuje lot”
- oprawnik → „ten, kto oprawia”

### -acz / -ącz
**Funkcja:** nomen agentis — wykonawca czynności (od czasownika)  
**Znaczenie:** „ten, kto wykonuje czynność X”  
**Typ:** deverbatywy  
**Geneza:** psł. *-ačь  

**Przykłady:**
- palacz → „ten, kto pali”
- biegacz → „ten, kto biega”
- pływacz → „ten, kto pływa”
- strzelacz → „ten, kto strzela”
- śpiewak → „ten, kto śpiewa”
- kopacz → „ten, kto kopie”
- kreślarz / kreślacz → „ten, kto kreśli”

### -arz / -erz
**Funkcja:** nomen agentis — wykonawca zawodu / rzemieślnik  
**Znaczenie:** „ten, kto zawodowo zajmuje się X / wyrabia X”  
**Typ:** denominalny i deverbatywy  
**Geneza:** psł. *-arjь, pożyczka z łaciny -ārius przez germański -āri  

**Przykłady:**
- lekarz → „ten, kto zajmuje się leczeniem”
- kucharz → „ten, kto gotuje”
- malarz → „ten, kto maluje”
- stolarz → „ten, kto robi stoły / meble”
- murarz → „ten, kto muruje”
- garncarz → „ten, kto robi garnki”
- kowal (< *kov-alь) → „ten, kto kuje” (wariant -al-)

### -ca / -wca
**Funkcja:** nomen agentis — sprawca / ten, kto wykonuje czynność  
**Znaczenie:** „ten, kto dokonuje X”  
**Typ:** deverbatywy (od form bezokolicznikowych)  
**Geneza:** psł. *-ьca / *-vьca  

**Przykłady:**
- śpiewca → „ten, kto śpiewa”
- twórca → „ten, kto tworzy”
- odbiorca → „ten, kto odbiera”
- dawca → „ten, kto daje”
- zbawca → „ten, kto zbawia”

### -tel / -iciel / -yciel / -aciel
**Funkcja:** nomen agentis — wykonawca czynności, często o wyższej godności lub trwałości  
**Znaczenie:** „ten, kto X (zawodowo / trwale / z godnością)”  
**Typ:** deverbatywy  
**Geneza:** psł. *-telь — pie. *-tēl  

**Przykłady:**
- nauczyciel → „ten, kto naucza”
- modliciel → „ten, kto modli się / wstawia”
- wybawiciel → „ten, kto wybawia”
- pocieszyciel → „ten, kto pociesza”

### -k / -ak / -ek / -ik / -ece / -ec / -iec / -ce + żeńskie: -ka / -aca / -eca / -ica / -ce
**Funkcja:** zdrobnienie / indywidualizacja / egzemplarz / nosiciel cechy / nomen iuvenile / feminatyw  
**Znaczenie:** „mały X / jeden egzemplarz X / młody X / ten którego definiuje X / żeńska wersja X / to co powstało przez X”  
**Typ:** denominalny, deverbatywy, deadiektywny  
**Geneza:** PIE *-(H)kos / *-ḱos — jeden morfem w ablautowych wariantach  

**Przykłady (-ek):**
- domek → „mały dom”
- kotek → „mały kot”
- orzeszek → „mały orzech”
- woreczek → „mały worek”

**Przykłady (-ik):**
- stolik → „mały stół”

**Przykłady (-ak, nosiciel cechy):**
- rybak → „ten którego domeną jest ryba”
- biedak → „ten którego definiuje bieda”
- strażak → „ten którego rolą jest straż”
- prostak → „ten który jest prosty”
- chłopak → „mały chłop / młody mężczyzna”

**Przykłady (-ak, nomen iuvenile):**
- szczeniak → „młody pies”
- źrebak → „młody koń”
- cielak → „młode cielę”
- dzieciak → „małe dziecko”

**Uwaga:** -ak jako nomen iuvenile konkuruje z -ię / -ęt-, ale z odcieniem potocznym i często odnosi się do osobnika już trochę podrosłego. -ę jest morfologicznie neutralne i prymarnie młode, -ak afektywne.

**Przykłady (-ec / -iec):**
- chłopiec → „mały chłop”
- starzec → „ten który jest stary”
- cudzoziemiec → „ten z cudzej ziemi”
- rybiec → „mały gatunek ryby”

**Przykłady (-ce / -ece):**
- serce → „mały środek”
- słońce → „małe słońce”
- jajce (dial.) → „małe jajo”

**Przykłady (wynik czynności):**
- znakъ → znak („to co zostało poznane”)
- zvǫkъ → dźwięk („to co zabrzmiało”)
- borkъ → wybór („to co zostało wzięte”)

### -ec
**Funkcja:** nomen agentis / nomen patientis / nomen rei — wykonawca lub przedmiot  
**Znaczenie:** „ten, kto X / ten, który pochodzi z X / ten, który posiada cechę X”  
**Typ:** denominalny i deverbatywy  
**Geneza:** psł. *-ьcь  

**Przykłady (osoby):**
- żołnierz (niem. pożyczka) / wojowiec → „ten, kto walczy”
- starzec → „ten, kto jest stary / starszy”
- nędzniec → „ten, kto jest nędzny”
- szlachcic → „ten, co jest szlachty”
- cudzoziemiec → „ten, kto jest z cudzej ziemi”

**Przykłady (rzeczy):**
- rybiec → „mały gatunek ryby”
- chłopiec → „mały chłop”

### -cierz/-torz/-cier (← PIE *-tḗr/-tōr) [~]
**Funkcja:** archaiczny formant nominalny o pierwotnie agentywno-funkcyjnym charakterze  
**Znaczenie:** „ten, który wykonuje czynność X / pełni funkcję X / jest związany rolą X wyrażoną przez rdzeń”  
**Typ:** odziedziczony morfem PIE; w prasłowiańszczyźnie nieproduktywny  

**Geneza:** PIE -tḗr (w systemie atematycznych formacji rzeczownikowych)  

**Przykłady (rekonstruowane):**
- pacierz → „ten, kto chroni” (*pH₂tḗr)
- macierz → „matka” (*méh₂tēr)
- bracierz → „brat” (*bʰréh₂tēr)
- dukcierz → „córka” (*dʰugh₂tḗr)
- ahcierz → „ten, kto jest gorący, suchy” (*h₂eh₁ter)
- gencierz → „ten kto produkuje, rodzi” (*ǵénh₁tōr)

### -dło / -tła / -sło / -sła / -śle / -tro / -tlo (← PIE *-tro- / *-tlo-)
**Funkcja:** archaiczny formant nominalny o charakterze instrumentalnym  
**Znaczenie:** „to, czym się wykonuje czynność X / to, co służy do X / mechanizm związany z rdzeniem X”  
**Typ:** odziedziczony morfem PIE  

**Przykłady (narzędzia/środki):**
- wiatr → (*u̯ē-tro-) „to, co wieje”
- żądło → (*źęd-dlo) „to, czym się żga / kłuje”
- szydło → (*šy-dlo) „to, czym się szyje”
- sidło → (*sě-dlo) „to, co służy do siadania (chwytania)”
- mydło → (*my-dlo) „to, co służy do mycia”
- radło → (*ordlo) „to, czym się orze”
- ciosła → (*tes-sla) „to, czym się cieśli”

**Przykłady (miejsca/wyniki):**
- piętro → (*pęt-tro) „stopień”
- gardło → (*gьr-dlo) „to, czym się pożera / graba”
- siodło → (*sed-lo) „to, na czym się siedzi”
- źródło → (*žer-dlo) „to, co wrze / wytryska”

### -uń / -un (psł. *-unь)
**Funkcja:** nomen agentis / hypocoristica — nosiciel cechy, często z odcieniem pieszczotliwym albo pejoratywnym  
**Znaczenie:** „ten, który ma cechę X / ten, który wykonuje X (z odcieniem afektywnym)”  
**Typ:** denominalny i deadiektywny  

**Przykłady:**
- śpiun → „ten, co śpi”
- gaduń → „ten, co gada”
- złodun (dial.) → „ten, co źle postępuje”
- ćpun → „ten, co ćpa”
- biegun → „ten, co biega / biegun jako punkt ruchu”
- pływun → „ten, co pływa”

### -taj/-otaj/-ataj (psł. *-tajь ← bsł. *-tājas ← PIE *-teh₂-i̯o-s)
**Funkcja:** nomen agentis — wykonawca czynności jako stały, powtarzający się jej podmiot  
**Znaczenie:** „ten, który X-uje często / stale / nałogowo”  
**Typ:** odrdzeniowy (deverbatywy)  

**Geneza:** PIE *-teh₂- → pierwotnie sufiks iteratywno-frekwentatywny czasowników; w słowiańskim uproszczony do *-tajь.  

**Przykłady bałtyckie:**
- lit. rašýtojas → „pisarz”
- lit. ganýtojas → „pasterz”
- lit. mókytojas → „nauczyciel”
- lit. ieškótojas → „poszukiwacz”
- lit. tárdytojas → „sędzia śledczy”
- łot. dziêdâtãjs → „śpiewak”

**Zachowane wyrazy słowiańskie:**
- rataj → „ten, który stale orze”
- szataj → „ten, który zajmuje się szatą”
- hultaj → „ten, który się włóczy”
- vozataj (cz. przestarz.) → „woźnica”

### -ię / -ę (psł. *-ę, temat *-ęt-)
**Funkcja:** nomen iuvenile — nazwa młodego / małego istnienia  
**Znaczenie:** „młody X / mały X / to, które dopiero jest”  
**Typ:** denominalny (od nazwy gatunku)  
**Geneza:** PIE *-ent- → psł. *-ęt- (przez nosówkę)  

**Przykłady:**
- jagnię
- źrebię
- kocię
- cielę
- prosię
- pisklę

### B. Kategoria abstrakcji / stanu / jakości

### -ość / -esc (psł. *-ostь)
**Funkcja:** reifikacja cechy / abstrakcja jakości przymiotnikowej  
**Znaczenie:** „stan bycia X / cecha X / właściwość X”  
**Typ:** deadiektywny  

**Przykłady:**
- młodość → „stan bycia młodym”
- starość → „stan bycia starym”
- ciemność → „stan bycia ciemnym”
- wolność → „stan bycia wolnym”
- grzeczność → „cecha bycia grzecznym”
- twardość → „właściwość twardości”
- mądrość → „cecha mądrości”
- przyjemność → „stan / jakość bycia przyjemnym”

### -ota / -ota (psł. *-ota)

## III. PRZYROSTKI TWORZĄCE CZASOWNIKI

### -ować
**Funkcja:** werbalizacja — tworzenie czasowników od rzeczowników i przymiotników  
**Znaczenie:** „działać w obrębie X / przynależeć funkcjonalnie do X / operować w domenie X / przejawiać X w działaniu”  
**Typ:** denominalny i deadiektywny → czasowniki imperfektywne  
**Geneza:** psł. -ovati ← -ov- + -a-ti  

**Przykłady:**
- pracować
- malować
- budować
- chorować
- ryzykować

### -ywać / -iwać (psł. *-yvati / *-ivati)
**Funkcja:** tworzenie wtórnych imperfektywów od czasowników dokonanych  
**Znaczenie:** „robić X wielokrotnie / robić X jako czynność powtarzającą się — niedokonany odpowiednik X”  
**Typ:** deverbatywy → od dokonanych, czasowniki imperfektywne  

**Przykłady:**
- napisać → napisywać
- rzucić → rzucać → zarzucać
- kupić → kupywać
- wyrabiać

### -tać / -otać / -atać (psł. *-tati / *-otati ← PIE *-teh₂-)
**Funkcja:** tworzenie czasowników iteratywno-frekwentatywnych  
**Znaczenie:** „robić X często / stale / z natury — nie jednorazowo, lecz habitualnie”  
**Typ:** odrdzeniowy (deverbatywy od rdzenia werbalnego)  

### -eć / -ieć (psł. *-ěti)
**Funkcja:** tworzenie czasowników stanowych / inchoacywnych  
**Znaczenie:** „stawać się X / wchodzić w stan X / być w stanie X”  
**Typ:** deadiektywny  

**Przykłady:**
- bieleć
- starzeć
- dojrzejeć
- twardnieć
- łagodnieć
- mądrzeć

### -szeć / -szać (psł. *-šati ← bsł. *-šatei ← PIE *-(h₁)seti)
**Funkcja:** tworzenie czasowników desideratywnych  
**Znaczenie:** „chcieć X / dążyć do X / być nastawionym na X”  
**Typ:** odrdzeniowy (od rdzenia werbalnego)  

### -ić / -yć (psł. *-iti)
**Funkcja:** werbalizacja kauzatywna / denominalna  
**Znaczenie:** „sprawić że X / czynić X / nadać właściwość X”  
**Typ:** denominalny, deadiektywny  

**Przykłady:**
- bielić
- czyścić
- gromadzić
- ciemnić
- zdobić
- wzbogacić
- uspokoić

### -nąć / -ną- (psł. *-nǫti)
**Funkcja:** tworzenie czasowników semelfaktywnych / inchoacywnych / momentalnych  
**Znaczenie:** „wykonać X jednorazowo / gwałtownie wejść w stan X”  
**Typ:** deverbatywy i deadiektywny  

**Przykłady:**
- krzyknąć
- stuknąć
- mrugnąć

- chałupuch (pej.)

### -eń / -ień (psł. *-enь / *-ьnь)
